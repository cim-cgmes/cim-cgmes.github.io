61970-452_Operation-AP-Con-Complex-SHACL

opc452.Accumulator​

Targets:

  • targetClass: cim.Accumulator
  • implicitClassTarget: opc452.Accumulator

Nested Properties:

opc452.Accumulator-accumulatorValues​

For Accumulator, Measurement.measurementType is restricted to the following valid values: ApparentEnergy, ReactiveEnergy, ActiveEnergy.

Messages:

  • "Not allowed value."

Constraints:

ComponentDetails
sh.InConstraintComponentValues: [ApparentEnergy, ReactiveEnergy, ActiveEnergy]

opc452.Accumulator-accumulatorUnitValues​

For Accumulator, Measurement.unitSymbol is restricted to the following valid values: VAh, VArh, Wh.

Messages:

  • "Not allowed value."

Constraints:

ComponentDetails
sh.InConstraintComponentValues: [cim.UnitSymbol.VAh, cim.UnitSymbol.VArh, cim.UnitSymbol.Wh]

opc452.Analog​

Targets:

  • targetClass: cim.Analog
  • implicitClassTarget: opc452.Analog

Nested Properties:

opc452.Analog-analogValues​

For Analog, Measurement.measurementType is restricted to the following valid values: ThreePhasePower, ThreePhaseActivePower, ThreePhaseReactivePower, LineCurrent, PhaseVoltage, Voltage, Angle, Frequency and TapPosition.

Messages:

  • "Not allowed value."

Constraints:

ComponentDetails
sh.InConstraintComponentValues: [ThreePhasePower, ThreePhaseActivePower, ThreePhaseReactivePower, LineCurrent, PhaseVoltage, Voltage, Angle, Frequency, TapPosition]

opc452.Analog-analogUnitValues​

For Analog, Measurement.unitSymbol is restricted to the following valid values: W, deg, VA, A, VAr, V, Hz.

Messages:

  • "Not allowed value."

Constraints:

ComponentDetails
sh.InConstraintComponentValues: [cim.UnitSymbol.W, cim.UnitSymbol.deg, cim.UnitSymbol.VA, cim.UnitSymbol.A, cim.UnitSymbol.VAr, cim.UnitSymbol.V, cim.UnitSymbol.Hz]
